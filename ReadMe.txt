Bienvenue dans le jeu du morpion !!!

Ici vous trouverez la procédure à suivre pour compiler et exécuter
notre fichier à la fois sous linux et windows

1- Sous LINUX:
	a-) S'assurer de la présence des fichiers suivants:
		- algominmax.c
		- ia.c
		- main.c 
		- makefile
		- menu.c
		- morpion.c
		- sauvegarde.c
		- stats.c
		- utilitaire.c 
		- morpion.h

	b-) Tapez la commande "make" dans un terminal pour compiler

	c-) Tapez ensuite la commande "./main" pour exécuter

	d-) Profitez de notre Jeu !!!

	e-) ATTENTION !!! Pour recompiler le programme supprimez d'abord 
		l'éxécutable grâce à la commande "rm main"

2- Sous WINDOWS:
	a-) Inclure tous ces fichiers dans un même projet sous codeblocks:
		- alominmax.c
		- ia.c
		- main.c 
		- makefile
		- menu.c
		- morpion.c
		- sauvegarde.c
		- stats.c
		- utilitaire.c 
		- morpion.h

	b-) Compilez et exécuter le projet

	c-) Profitez de notre Jeu !!!
